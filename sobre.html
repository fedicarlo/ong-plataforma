ONG Unidas Pela Paz

Este site foi desenvolvido como projeto acadêmico com o objetivo de criar a página institucional de uma ONG fictícia chamada Unidas Pela Paz. O foco é demonstrar domínio das tecnologias front-end básicas e a organização de um site completo, responsivo e sem dependência de frameworks.

Funcionalidades

  • Página inicial com banner, contadores e destaques de projetos
  • Seção de causas e campanhas com cards (projetos)
  • Páginas individuais para cada projeto (suporte, psicológico, atividades)
  • Formulário simples para cadastro/voluntariado
  • Página Projetos listando todas as ações em andamento
  • Página Sobre (esta) com descrição do trabalho

Tecnologias utilizadas

  • HTML5
  • CSS3 (arquitetura por módulos: tokens, base, layout, components, menu, forms e responsive)
  • JavaScript (menu e modal simples)
  • Google Fonts (Manrope)
  • Imagens locais em assets/images

Objetivo do projeto

Demonstrar boas práticas de estruturação, semântica, responsividade e organização de estilos, resultando em um site claro, acessível e fácil de manter.

Como rodar

  1. Abrir o arquivo index.html no navegador
  2. Arquivos organizados em css/ e assets/
