Unidas Pela Paz — Projeto acadêmico

Objetivo:
Demonstrar domínio de HTML5, CSS3 e JavaScript na construção de um site institucional responsivo para uma ONG fictícia.

Funcionalidades:
- Home com banner, contadores, cards de projetos e seção “Do blog/Eventos” estática
- Página “Projetos” com cards e links para páginas individuais
- Páginas de projeto: Suporte Alimentar, Apoio Psicológico, Atividades Infantis
- Formulário simples para voluntários (cadastro.html)
- Página “Sobre” com descrição do trabalho, tecnologias e como rodar

Tecnologias:
- HTML5, CSS3 modular (tokens, base, layout, components, menu, forms, responsive), JS simples
- Google Fonts (Manrope)
- Imagens locais em assets/images

Estrutura:
index.html, projetos.html, cadastro.html, sobre.html,
projeto-suporte.html, projeto-psicologico.html, projeto-atividades.html,
css/, assets/
